About

Agent-first measurement and instrumentation for AI models. Pricing, events, and history exposed as JSON-first APIs that agents can call directly. Humans get a dashboard built on the same endpoints.

Data is community-maintained, source-cited, and committed to a public GitHub repo. Every entry carries a verification date so callers can judge freshness.
