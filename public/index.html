modelmeter

Agent-first cost calculator for AI models. Built for machines, friendly to humans.

What this is: a stable, machine-callable API for LLM cost estimation, plus this human-friendly calculator. Every endpoint returns JSON. Every URL is documented in the OpenAPI spec.

Pricing data is in a public GitHub repo with daily snapshots. Found a price change? Open a PR.
