About

The changelog of record for the AI stack. Deprecations, sunsets, price changes, and releases in the model layer — recorded as source-verified events with dates, migration targets, and verbatim quotes. Agents call the same JSON endpoints this dashboard is built on.

Every event cites its source, most with archive snapshots. Entries are published fast as unverified, then human-reviewed to verified — corrections are never silent. The full dataset lives in a public GitHub repo.
